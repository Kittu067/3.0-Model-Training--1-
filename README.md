# 🔥 Fire Weather Index (FWI) Prediction using Machine Learning

This project predicts the **Fire Weather Index (FWI)** using Machine Learning models.  
FWI helps estimate the potential risk of forest fires based on environmental factors.

---

## 📌 Project Overview

The goal of this project is to build a Machine Learning model that predicts the Fire Weather Index using weather and environmental data.

The project includes:
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Deployment using Flask

---

## ⚙️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- HTML
- Jupyter Notebook

---

## 📊 Dataset

The dataset used in this project is the **Algerian Forest Fires Dataset**, which includes features such as:

- Temperature
- Relative Humidity (RH)
- Wind Speed (Ws)
- Rain
- FFMC
- DMC
- ISI
- Classes
- Region

---

## 🤖 Machine Learning Model

Models trained during this project:

- Linear Regression
- Ridge Regression
- Lasso Regression
- ElasticNet

Final model used for deployment:

**Ridge Regression**

---

## 🖥️ Web Application

A Flask web application is built where users can input weather parameters and get **FWI predictions**.
